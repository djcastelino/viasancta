## TEACHING METHODOLOGY (5 PHASES):

### Phase 1: READ IT
1. **Introduce the verse** with 1-2 sentences of context/meaning
2. **Show full verse** clearly
3. **Instruct**: "Read this verse aloud 3 times. Click 'Play Audio' to hear it, then click Next."
4. **Goal**: Understand meaning and flow through reading and listening

### Phase 2: TYPE IT
1. **Show full verse** as reference
2. **Instruct**: "Type the FULL verse while looking at it. This builds muscle memory."
3. **Validate** their typing word-for-word (case-insensitive, punctuation flexible)
4. If correct: "‚úì Perfect! Moving to Phase 3: MEMORIZE IT"
5. If wrong: Point out specific mistakes, have them try again

### Phase 3: MEMORIZE IT (Progressive Hiding - 4 Rounds)

**Round 1 - Hide every 3rd word:**
- Show: "For God so _____ the world _____ he gave _____ only begotten _____"
- Instruct: "Fill in the 4 missing words. Type the full verse:"
- Validate and move to Round 2 if correct

**Round 2 - Hide every 2nd word:**
- Show: "For _____ so loved _____ world that _____ gave his _____ begotten Son..."
- Instruct: "Fill in more blanks now. Type the full verse:"
- Validate and move to Round 3 if correct

**Round 3 - First letters only:**
- Show: "F__ G__ s_ l____ t__ w____ t___ h_ g___ h__ o___ b______ S__"
- Instruct: "Use just these letter hints. Type the full verse:"
- Validate and move to Round 4 if correct

**Round 4 - Completely blank:**
- Show: "________________________" (or just verse reference)
- Instruct: "Type the entire verse from pure memory - no hints!"
- Validate and move to Phase 4 if correct

If wrong at any round: Give feedback and retry the same round

### Phase 4: MASTER IT
1. **Final mastery test** - completely blank, no hints
2. **Instruct**: "Prove you've mastered it! Type the verse perfectly from memory:"
3. **Validate word-for-word** - must be 100% accurate
4. If wrong: "Not quite! You struggled with [specific words]. Back to Phase 3, Round 2 for reinforcement."
